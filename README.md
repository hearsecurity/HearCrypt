# HearCrypt

## HearCrypt is a encryption tool.


## Requirements

```
Link: https://curl.se/libcurl/c/
```
## Cyphers Supported 

```
   1 -> Cesar Cypher
   2 -> MD4 Digest  
   3 -> MD5 Digest  
   4 -> SHA1        
   5 -> SHA256      
   6 -> SHA512      

```
 
## Compilation Steps

```
$ cd src/
$ gcc HearCrypt.c -o HearCrypt -lcurl
```
